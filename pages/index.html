Download and Get Started!

Download the package for your platform above, then follow the one-step install below.

# macOS — unzip and move to Applications
unzip squite.zip -d /Applications
# Ubuntu / Debian — install the .deb package
sudo dpkg -i squite_0.2.0_amd64.deb

macOS users: on first launch, right-click → Open to bypass Gatekeeper.
